## 📌 About

This repository documents my hands-on learning in **Web Application Security Testing** through labs from PortSwigger Web Security Academy.

The goal is to understand vulnerabilities, exploit them in a controlled lab environment, and study mitigation techniques.

## 🛠 Tools

* Burp Suite
* Browser Developer Tools
* Manual HTTP Analysis

* Cross-Site Scripting (XSS)
* OS Command Injection
* Path Traversal
* Authentication Vulnerabilities
* Access Control Vulnerabilities
